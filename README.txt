GentleDB for Qt.

License: LGPL 2.1+

Requirements:

* Qt 4.7            <qt.nokia.com>
* QCA               <delta.affinix.com/qca>
* QCA OSSL          <delta.affinix.com/qca>

Compile thusly:

    qmake && make

Install thusly:

    sudo make install
    sudo ldconfig  # only on Linux

Use in your *.pro file thusly:

    LIBS += -lqtgentledb
